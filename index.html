This project was the first coding project I ever did, it was done as part of the ONE program in format of a challenge. It is basically a cipher (both an encoder and a decoder). You insert a text you want to encrypt, and than the program returns the encrypted format.
This program also implements some restrictions, e.g. you may not insert uppercase letters. And it has a button to copy the generated content.
